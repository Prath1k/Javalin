# PixelArena - Minimalist Game Hub

A premium, Scandinavian Minimalist / Clean Tech game hub featuring a collection of 12 highly polished, interactive games.

## 🕹️ Game Collection
1.  **Pac-Man**: Classic maze action with modern visuals.
2.  **Space Invaders**: Retro shooter with neon effects.
3.  **Bottle Spin 2.0**: Spin the bottle with microphone-blow interaction.
4.  **Neon Snake**: Retro snake with speed HUD and fluid movement.
5.  **Glass Memory**: Memory matching with a clean glassmorphism aesthetic.
6.  **Pulse Rhythm**: Beat-based clicking game with visualizers.
7.  **Terminal Hacker**: Matrix-style typing challenge with a WPM counter.
8.  **Quantum Sweeper**: Modern take on Minesweeper.
9.  **Neon Pong**: Classic Pong with AI and screen shake effects.
10. **Sort Viz**: Interactive sorting algorithm visualization.
11. **Hex Connect**: Strategic connection game.
12. **Grandmaster Chess**: Full-featured chess with a sleek UI.

## ✨ Features
- **Global Scrolling**: Fully responsive across all devices and screen sizes.
- **Light/Dark Mode**: Dynamic theme switching with custom branding.
- **Premium UI**: Micro-animations, particle effects, and high-fidelity transitions.
- **Online Multiplayer**: Real-time multiplayer support for selected games (e.g., Tic-Tac-Toe).

## 🚀 Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
